# 版本历史查看功能 - 快速演示

## 🎬 5分钟快速演示

这是一个快速演示指南，帮助您在5分钟内体验版本历史查看功能的所有核心特性。

---

## 📋 准备工作（1分钟）

### 1. 确保服务运行
```bash
# 检查后端服务
docker ps | findstr operation-analysis-v2-backend

# 检查前端服务（应该在 http://localhost:5173）
```

### 2. 打开浏览器
访问：http://localhost:5173

### 3. 准备测试数据
确保您有：
- ✅ 至少1个会话
- ✅ 该会话至少有2个版本（V1, V2）

如果没有，请先：
1. 创建新会话
2. 上传Excel文件
3. 生成报告
4. 点击"保存版本"按钮，保存为V1
5. 修改报告内容（使用AI对话）
6. 再次点击"保存版本"，保存为V2

---

## 🎯 演示步骤（4分钟）

### 步骤1：展开版本面板（30秒）

1. **找到会话项**
   - 在左侧会话列表中，找到有版本的会话
   - 会话项右侧有一个 📄 图标

2. **点击图标**
   - 点击 📄 图标
   - 观察版本面板平滑展开

3. **查看版本列表**
   - 版本按时间倒序排列（最新的在上面）
   - 当前版本有绿色"[当前]"标签
   - 每个版本显示：版本号、说明、时间

**预期效果：**
```
┌─────────────────────────────────────┐
│ 📄 LTV - 副本 (2)                    │
│    今天                    [📄]      │  ← 点击这里
│    进行中                            │
├─────────────────────────────────────┤
│ 📌 版本历史 (2个版本)               │  ← 面板展开
│ ┌─────────────────────────────────┐ │
│ │ V2 - 添加了新的分析章节          │ │
│ │ 2分钟前              [当前]     │ │  ← 当前版本
│ └─────────────────────────────────┘ │
│ ┌─────────────────────────────────┐ │
│ │ V1 - 初始版本                   │ │
│ │ 10分钟前                        │ │  ← 历史版本
│ └─────────────────────────────────┘ │
└─────────────────────────────────────┘
```

---

### 步骤2：查看历史版本（1分钟）

1. **点击历史版本**
   - 点击版本列表中的 V1

2. **观察加载过程**
   - 显示提示："正在加载版本 V1..."
   - 右侧报告内容开始切换

3. **查看版本提示条**
   - 页面顶部出现蓝色提示条
   - 显示："正在查看历史版本 V1 - 初始版本 (10分钟前)"
   - 右侧有"返回当前版本"按钮

4. **查看报告内容**
   - 右侧显示V1版本的完整报告
   - 包括文字和图表

**预期效果：**
```
┌──────────────────────────────────────────────────────────┐
│ ℹ️ 正在查看历史版本 V1 - 初始版本 (10分钟前)  [返回当前版本] │  ← 版本提示条
└──────────────────────────────────────────────────────────┘

右侧显示V1版本的报告内容...
```

---

### 步骤3：对比不同版本（1分钟）

1. **记住V1的内容**
   - 快速浏览V1的报告内容
   - 注意关键数据和图表

2. **切换到V2**
   - 在左侧版本列表中点击 V2
   - 观察报告内容的变化

3. **对比差异**
   - V2可能有新增的章节
   - V2可能有修改的数据
   - V2可能有优化的图表

**提示：**
- 可以多次在V1和V2之间切换
- 每次切换都会显示加载提示和成功消息

---

### 步骤4：返回当前版本（30秒）

1. **点击"返回当前版本"按钮**
   - 在版本提示条中点击按钮

2. **观察变化**
   - 版本提示条消失
   - 报告内容恢复为最新版本
   - 显示成功消息："已返回当前版本"

3. **验证状态**
   - 左侧版本列表中，当前版本仍有"[当前]"标签
   - 右侧显示最新的报告内容

**预期效果：**
```
✅ 已返回当前版本

版本提示条消失，报告内容恢复为V2（当前版本）
```

---

### 步骤5：折叠版本面板（30秒）

1. **再次点击 📄 图标**
   - 版本面板平滑收起

2. **验证状态**
   - 版本面板已隐藏
   - 会话列表恢复正常显示

---

## 🎨 交互细节演示（1分钟）

1. 将鼠标悬停在版本项上
2. 观察背景色变化（变为浅蓝色）
3. 观察鼠标指针变化（变为手型）

### 加载状态
1. 点击版本时观察加载动画
2. 旋转图标 + "正在加载版本..."文字

### 成功提示
1. 切换版本后观察成功消息
2. 绿色提示："已切换到版本 VX"
