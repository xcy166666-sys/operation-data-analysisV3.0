# 图表AI编辑功能 - 快速开始

## 🚀 5分钟快速体验

### 第1步：启动系统（1分钟）

**启动后端：**
```bash
cd backend
.\start_local.bat
```

**启动前端：**
```bash
cd frontend
npm run dev
```

**登录系统：**
- 访问：http://localhost:5173
- 用户名：`admin`
- 密码：`admin123!`

---

### 第2步：生成测试报告（2分钟）

1. 点击"数据分析"进入分析页面
2. 上传Excel文件（可以使用 `backend/test_data.xlsx`）
3. 输入分析需求：
   ```
   生成一份用户活跃度分析报告，包含柱状图展示每日活跃用户数
   ```
4. 点击"提交生成报告"
5. 等待报告生成完成（约30秒）

---

### 第3步：体验图表编辑（2分钟）

#### 方式1：快捷编辑（推荐新手）

1. **悬停图表**
   - 将鼠标移动到图表上
   - 看到底部工具栏滑入
   - 首次使用会看到提示："💡 点击可以编辑这个图表"

2. **改颜色**
   - 点击"改颜色"按钮
   - 选择一个喜欢的颜色（如绿色）
   - 点击"应用修改"
   - 等待5秒，图表颜色改变！

3. **换类型**
   - 点击"换类型"按钮
   - 选择"折线图"
   - 点击"应用修改"
   - 图表从柱状图变成折线图！

#### 方式2：AI自由修改（推荐高级用户）

1. **打开AI编辑**
   - 悬停图表，点击"AI修改"按钮
   - 或直接点击图表，然后点击"AI修改"

2. **输入修改指令**
   - 在快捷编辑面板的"AI修改"区域输入：
   ```
   改成渐变色，从蓝色到绿色，并添加数据标签
   ```
   - 点击"应用修改"

3. **查看结果**
   - 等待10-15秒
   - AI理解你的指令并重新生成图表
   - 图表变成渐变色，并显示数据标签！

#### 方式3：AI对话模式（最灵活）

1. **进入对话模式**
   - 点击右上角"AI对话"按钮
   - 对话面板在左侧打开

2. **对话修改图表**
   - 在对话框输入：
   ```
   把图表改成橙色主题，并调整高度为800px
   ```
   - 发送消息
   - AI理解并修改图表

3. **连续修改**
   - 继续输入：
   ```
   再添加一个平滑曲线效果
   ```
   - AI会在之前的基础上继续修改

## 💡 使用技巧

### 1. 颜色修改技巧
- **预设颜色**：快速选择8种常用颜色
- **自定义颜色**：输入颜色代码，如 `#409eff`
- **渐变色**：使用AI指令，如"改成从蓝色到绿色的渐变"

### 2. 类型切换技巧
- **柱状图**：适合对比数据
- **折线图**：适合展示趋势
- **饼图**：适合展示占比

### 3. AI指令技巧

**简单指令：**
- "改成蓝色"
- "换成折线图"
- "调整高度为700px"

**复杂指令：**
- "改成渐变色，从蓝色到绿色，并添加数据标签"
- "使用深色主题，背景色为深灰色，文字为白色"
- "添加平滑曲线效果，并显示数据点"

**组合指令：**
- "改成橙色主题的折线图，添加数据标签和网格线"
- "使用渐变色，调整图表大小为宽度100%高度800px，添加标题"

## 🎯 常见场景示例

### 场景1：快速改颜色
**需求：** 图表颜色不符合品牌色

**操作：**
1. 悬停图表 → 点击"改颜色"
2. 输入品牌色代码：`#FF6B6B`
3. 应用修改

**耗时：** 10秒

---

### 场景2：切换图表类型
**需求：** 柱状图改成折线图更清晰

**操作：**
1. 悬停图表 → 点击"换类型"
2. 选择"折线图"
3. 应用修改

**耗时：** 10秒

---

### 场景3：复杂样式调整
**需求：** 需要渐变色、数据标签、调整大小

**操作：**
1. 悬停图表 → 点击"AI修改"
2. 输入指令：
   ```
   改成从蓝色到紫色的渐变，添加数据标签，
   调整高度为800px，使用圆角效果
   ```
3. 应用修改

**耗时：** 15秒

---

### 场景4：对话式连续修改
**需求：** 需要多次调整直到满意

**操作：**
1. 打开AI对话面板
2. 第一次："改成蓝色主题"
3. 第二次："再添加数据标签"
4. 第三次："调整图表高度为700px"
5. 第四次："使用平滑曲线"

**耗时：** 每次10-15秒

## ❓ 常见问题

### Q1：为什么悬停没有显示工具栏？
**A：** 
- 确认图表已经生成完成
- 刷新页面重试
- 检查浏览器控制台是否有错误

### Q2：修改后图表没有变化？
**A：**
- 检查网络连接
- 查看是否有错误提示
- 等待更长时间（AI处理需要10-15秒）
- 刷新页面重新生成报告

### Q3：AI不理解我的指令？
**A：**
- 使用更简单明确的语言
- 参考上面的指令示例
- 分步骤修改，不要一次性要求太多
- 使用快捷编辑代替AI指令

### Q4：修改后数据变了？
**A：**
- 这是Bug，请报告给开发团队
- AI应该只修改样式，不改变数据
- 刷新页面恢复原始图表

### Q5：可以撤销修改吗？
**A：**
- 当前版本暂不支持撤销
- 可以刷新页面恢复原始图表
- 或者使用AI指令："恢复原始样式"
